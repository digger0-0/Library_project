Library Collection Classification

Hong Kong Public Libraries adopt the "A System of Book Classification for Chinese Libraries" by LIU Guojun and “Dewey Decimal Classification” respectively for the classification of Chinese and English library materials. If you want to browse the library collection by classification number, you may refer to the classification tables below for reference.
